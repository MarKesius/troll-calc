# 🤡 Troll Calculator

A calculator that looks perfectly normal... but has a mind of its own.

## What is this?

It's a classic Windows 11-style calculator with one small twist: **there's a 100% chance it will secretly perform a different operation than the one you pressed. Every. Single. Time.**

Press `+`? Maybe it does `×`.  
Press `−`? Maybe it does `÷`.  
Press `÷`? Maybe it does `+`.  

You'll never know until you see the result. 🎪

## Features

-  Pixel-perfect Windows 11 Calculator dark theme
-  Full keyboard support
-  Memory functions (MC, MR, M+, M−, MS)
-  100% troll chance on every operation — guaranteed chaos
-  Floating emoji reactions when trolled
-  Sarcastic Greek toast messages
-  Smooth animations and ripple effects

## Tech Stack

- Pure HTML, CSS, JavaScript
- No frameworks, no dependencies
- Static site — deploys anywhere
